To build:

mvn clean package

To run:

java -jar target\daysbetween-0.0.1-SNAPSHOT.jar

Example usage:

C:\git\daysbetween>java -jar target\daysbetween-0.0.1-SNAPSHOT.jar
02/06/1983 22/06/1983
Output(Days): 19
04/07/1984 25/12/1984
Output(Days): 173
03/01/1989 03/08/1983
Output(Days): 1979
